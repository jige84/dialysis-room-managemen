# 血液透析系统 - 部署指南

## 一、MCP 配置（已完成）

配置文件已创建在：`C:\Users\你的用户名\.cursor\mcp.json`

### 你需要做的：

1. **打开配置文件**
   - 按 `Win + R`，输入：`notepad %USERPROFILE%\.cursor\mcp.json`
   - 点击确定

2. **修改密码**
   - 找到：`"SSH_PASSWORD": "YOUR_PASSWORD_HERE"`
   - 将 `YOUR_PASSWORD_HERE` 替换为你的 ECS root 密码
   - 例如：`"SSH_PASSWORD": "abc123456"`

3. **保存并关闭**
   - 按 `Ctrl + S` 保存
   - 关闭记事本

4. **重启 Qoder**
   - 完全关闭 Qoder
   - 重新打开 Qoder

## 二、自动部署脚本使用方法

### 方式一：使用 WinSCP + 手动执行（推荐，最简单）

#### 第 1 步：下载 WinSCP
- 访问：https://winscp.net/
- 下载并安装

#### 第 2 步：连接 ECS
1. 打开 WinSCP
2. 新建会话：
   - 文件协议：SFTP
   - 主机名：`47.114.111.216`
   - 用户名：`root`
   - 密码：你的密码
3. 点击登录

#### 第 3 步：上传文件
1. 在右侧（远程）进入 `/opt` 目录
2. 在左侧（本地）找到你的项目文件夹 `E:\xuetoushiguanli`
3. 将整个文件夹拖到右侧 `/opt` 目录下
4. 等待上传完成

#### 第 4 步：执行部署
1. 打开 PuTTY 或 Windows Terminal
2. 连接 ECS：`ssh root@47.114.111.216`
3. 输入密码登录
4. 执行以下命令：

```bash
cd /opt/xuetoushiguanli
chmod +x deploy-to-ecs.sh
bash deploy-to-ecs.sh
```

5. 等待部署完成（约 5-10 分钟）

---

### 方式二：使用命令行一键部署（适合技术人员）

#### 前置要求
安装 PowerShell SSH 模块（只需执行一次）：

```powershell
# 以管理员身份运行 PowerShell
Install-Module -Name Posh-SSH -Force
```

#### 执行部署

```powershell
cd E:\xuetoushiguanli
.\deploy-helper.ps1
```

然后按提示输入密码即可。

## 六、故障排查

### 如果部署失败：

1. **检查端口是否开放**
   - 登录阿里云控制台
   - 进入安全组配置
   - 确认 22、80、443、3080 端口已开放

2. **检查服务状态**
   ```bash
   pm2 status
   systemctl status nginx
   systemctl status postgresql
   systemctl status redis
   ```

3. **查看错误日志**
   ```bash
   pm2 logs
   cat /var/log/hemodialysis-err.log
   ```

4. **手动启动服务**
   ```bash
   cd /opt/hemodialysis/backend
   npm start
   ```

## 七、安全建议

1. **修改 SSH 默认端口**（可选）
   ```bash
   # 编辑 SSH 配置
   nano /etc/ssh/sshd_config
   
   # 修改 Port 22 为其他端口，如 2222
   Port 2222
   
   # 重启 SSH
   systemctl restart sshd
   ```

2. **配置防火墙**
   ```bash
   ufw allow 80
   ufw allow 443
   ufw allow 2222  # 如果修改了 SSH 端口
   ufw enable
   ```

3. **定期备份**
   ```bash
   # 数据库备份
   pg_dump -U hd_app hemodialysis_db > backup-$(date +%Y%m%d).sql
   ```
