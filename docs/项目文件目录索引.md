# 血液透析室管理系统 — 项目文件目录索引

> 涉县善谷医院血液透析室 · 前后端分离（Express + PostgreSQL + React/Vite）  
> 本索引与源码文件头部的 JSDoc 注释对应，便于按模块查阅职责。

## 一、仓库根目录

| 路径 | 说明 |
|------|------|

| `backend/migrations/` | PostgreSQL 迁移脚本（表结构演进） |

### 2.4 中间件 `backend/src/middleware/`

| 文件 | 主要功能 |
|------|----------|

| `nurseTimeRestriction.js` | 护士仅可录入当班日期透析记录等业务限制 |

## 五、维护说明

- 新增源码文件时：请在文件首行起添加 **中文 JSDoc 文件头**（说明文件职责、主要功能、与上下游模块关系），并在 **主要导出函数/路由分组** 处补充简短段落注释。
- 权限与安全：路由须 `authenticate` + `authorize`/`rbac`；SQL 参数化；响应不含加密字段明文；日志不含患者姓名与证件号。
